

To run the Flask side:
Navigate to \WitnessStatementProcessing\QandA
run the command 'flask run'

To run the react app:
Open a new terminal, seperate from the flask terminal
Navigate to \WitnessStatementProcessing\FrontEnd\reactapp
run the command 'npm install --legacy-peer-deps' to install the react app
run the command 'npm start'
